SILENT HILL 2 REMAKE FAN PAGE

Este fue uno de mis primeros proyectos, inspirado en la saga Silent Hill 2 Remake, lanzada para diversas plataformas de videojuegos, tanto en consolas como en PC. Lo desarrollé con dedicación y esfuerzo, aplicando mis conocimientos básicos en HTML, CSS y JavaScript. Descripción: Este proyecto es una fan page dedicada a Silent Hill 2, donde los usuarios pueden explorar contenido relacionado con el juego, incluyendo noticias, imágenes y foros de discusión. La página está diseñada para ser visualmente atractiva y fácil de navegar, utilizando tecnologías web modernas para ofrecer una experiencia de usuario fluida.

Plataforma Web de Sushi

Este proyecto fue desarrollado para clientes que administran locales de comida japonesa, negocios de comida rápida o ventas de productos. Incluye una sección de inventario que permite gestionar información clave, como nombre, código, precio y descripción de los productos. Descripción: Esta plataforma web de sushi permite a los usuarios realizar pedidos en línea, gestionar su menú y llevar un control de inventario. Además, cuenta con una interfaz intuitiva que facilita la navegación y mejora la experiencia del cliente. Los administradores pueden actualizar fácilmente la información de los productos y recibir notificaciones sobre el estado de los pedidos, lo que optimiza la operación del negocio.
